# Visualization Project - Phase 2

Dies ist ein neues Vite-Projekt für D3-basierte journalistische Datenvisualisierungen zur Bundestags-Wahl.

## Projektstruktur

- **src/**: Quellcodedateien (main-stable.js, main-test.js)
- **Dok/**: Dokumentation (visualisierungs-kontext.md)
- **.github/**: GitHub-spezifische Dateien (copilot-instructions.md)
- **.vscode/**: VS Code-Einstellungen
- **index-stable.html**: Einstiegspunkt für stabile Version
- **index-test.html**: Einstiegspunkt für Test-/Experimentalversion
- **changelog.md**: Dokumentation aller Änderungen
- **prompt-log.md**: Protokoll aller KI-Interaktionen

## Verwendung

### Stabile Version

Öffnen Sie `index-stable.html` im Browser oder verwenden Sie Live Preview.

### Test/Experimentalversion

Öffnen Sie `index-test.html` im Browser oder verwenden Sie Live Preview.

## Abhängigkeiten

Das Projekt ist vorbereitet für:

- D3.js für Datenvisualisierungen
- Vite als Build-Tool

## Dokumentation

Konsultieren Sie `Dok/visualisierungs-kontext.md` für den konzeptionellen Rahmen des Projekts sowie `.github/copilot-instructions.md` für die Arbeitsanweisungen zum GitHub Copilot.
